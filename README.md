## 🧐 Features

- **JavaDoc**

- **Java ORM and DAO layer**

- **SpringBoot**

- **Units Test**
- **DTO Mapper**
- **Services**

## 🛠️ Installation Steps

1. Clone the repo

```bash
git clone https://github.com/minesenzo/Middleware_Project
```

2. Change directory

```bash
cd Middleware_Project
```

3. Install dependencies

```bash
sudo apt-get update
sudo apt-get install openjdk-11-jdk maven
mvn install package
```

5. Start the projet

```bash
mvn test
```
